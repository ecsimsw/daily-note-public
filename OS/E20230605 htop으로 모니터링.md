## htop으로 모니터링.md

### Installation
```
sudo apt update && sudo apt upgrade
sudo apt install htop
```

### Usage

- Cpu Core 별 사용량
- Memory, Swap 사용량
- tasks : 프로세스 수, thr : 스레드 수, running : 실행 중인 스레드 수
- Load average : 1분, 5분, 15분 동안의 평균 부하량
