## drwxr-xr-x 4 mark staff 128 4 3 0066.

#### 파일 타입
```
'd' : dir 
'-' : 일반파일
```

## 권한 정보
```
rwxr-xr-x : 권한 정보

r(읽기): 파일의 읽기 권한
w(쓰기): 파일의 쓰기 권한
x(실행): 파일의 실행 권한
````

```
rwx : 소유자 
r-x : 그룹 사용자 
r-x : 모든 사용자
```

## chmod 
rwxr-xr-x : rwx(7) + r-x(5) + r-x(5)

```
chmod 755 {FILE}

chmod 755 -R {DIR}
```
